Curriculum

Javascript y MySQL

Universidad de La Salle 2021

Mi introducción a la programación se produjo en la segunda mitad de 2021, cuando cursaba el cuarto semestre de mi carrera en ingeniería en automatización. En ese momento, cursé una materia llamada "Programación Aplicada", que me sumergió en el mundo de Java y la programación orientada a objetos. Además, en este curso adquirí conocimientos sobre MySQL y bases de datos.

Html y CSS

Universidad de La Salle 2021

Más adelante, continué con otra asignatura denominada "Sistemas de Información". En este curso, se profundizó en el desarrollo de páginas web, utilizando tecnologías como HTML y CSS. Durante esta materia, me sumergí en la creación de diversas landing pages, lo que me permitió fortalecer y expandir mis habilidades en estas tecnologías.

React y Firebase

Universidad de La Salle 2022

En el octavo semestre de mi carrera, inicié la elaboración de mi proyecto de grado. Este consistió en el desarrollo de un sistema de información web diseñado para facilitar el acceso, seguimiento y administración de pacientes y terapeutas en la clínica de optometría de la Universidad de La Salle. Para llevar a cabo este proyecto, se empleó React como framework y Firebase como plataforma de base de datos.

GitHub

Universidad de La Salle 2022

En la elaboración de mi proyecto de grado, colaboré con un compañero de mi carrera. Esto requirió que ambos adquiriéramos habilidades en el uso de GitHub para coordinar nuestro trabajo de codificación de manera efectiva. Aprendí a crear repositorios, realizar commits para rastrear las modificaciones realizadas y a trabajar en ramas separadas para gestionar el desarrollo de manera más eficiente.

C

Universidad de La Salle 2023

Durante mi noveno semestre, cursé a una materia denominada "Sistemas Digitales", donde adquirí conocimientos sobre programación elemental en C. Como la manipulación de arrays, la creación de funciones, el uso de estructuras condicionales y más. Además, durante este curso, aprendí a utilizar el software CodeWarrior, el cual se empleó para programar máquinas de estados finitos en un microcontrolador.

Wordpress

A mitad del 2023, conocí la plataforma WordPress, la cual me pareció muy interesante ya que permite crear páginas web de manera muy intuitiva y rápida. Me la pasé unas cuantas semanas testeando y creando páginas no muy complicadas, pero con el objetivo de aprender y entender bien cómo funciona esta herramienta.
